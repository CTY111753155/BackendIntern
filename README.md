### 取得專案

```bash
git clone https://github.com/CTY111753155/BackendIntern.git
```

### 安裝套件

```bash
npm install
```
### 運行專案

```bash
npm start
```
### 開啟專案

在瀏覽器網址列輸入以下即可看到畫面

```bash
http://localhost:3000/

```
### 使用匯率計算

在瀏覽器網址列輸入以下即可看到畫面

```bash
http://localhost:3000/exchange?source=USD&target=JPY&amount=$1,525

```

## 專案技術

- Node.js 
- Express
